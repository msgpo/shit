# printf escape code colors

```
# escape code chars relevant to text effects

    1) ;; # bold
    2) ;; # dark colors
    3) ;; # bright colors
    4) ;; # underline
    5) ;; # blink
    6) ;; # blink fast
    7) ;; # inverted
    8) ;; # hidden

esac
```
